# Cop4331-Java-Project
Fitness Tracker Application
---------------------------

Run Instructions:
1. Download app.jar
2. Open a terminal
3. Run: java -jar app.jar
4. Application starts with MainView

Features:
- Add/edit/delete workouts
- View and filter history
- Set goals
- View progress summary
- Save/load data
- Export to CSV

Requirements:
- Java SE 8 or higher
